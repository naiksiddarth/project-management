# Project Management API

A backend learning project built to understand core concepts of server-side development, REST APIs, and database operations.

## Purpose

This project serves as a hands-on learning experience for building a project management system backend from scratch. It focuses on understanding:
- API design and HTTP methods
- Request/response handling
- Data management and storage
- Server-side logic and business rules

## Getting Started

### Prerequisites
- Node.js installed on your machine

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

```bash
npm test
```
